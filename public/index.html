🧪 Guia do Avaliador: Como testar os cenários

Cenário 1: Concorrência Real (Prevenção de Duplicidade)

  • Clique no botão verde "🔄 Gerar Nova Chave" para garantir uma requisição Nova.
  • Em seguida, clique direto no botão vermelho "🔥 Disparar 3x Simultâneas".
  • O que observar: O banco de dados travará duas requisições instantaneamente (409 Conflict). A requisição "vencedora" processará por ~3 segundos e retornará 200 OK.

Cenários 2 e 3: Retry Seguro após Sucesso/Falha

  • Com o processamento anterior concluído, não troque a chave.
  • Clique no botão azul "▶ Enviar 1 Requisição Normal".
  • O que observar: Não haverá reprocessamento (sem delay). A API retornará 200 OK instantaneamente com o mesmo status salvo no banco.
